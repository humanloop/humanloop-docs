Humanloop supports all the major large language model providers, including OpenAI, Anthropic, Google, Azure, and more. Additionally, you can use your own custom models with Humanloop with the API.

## Providers

Here is a summary of which providers we support and whether

| Provider  | Models | Cost information | Token information |
| --------- | ------ | ---------------- | ----------------- |
| OpenAI    | ✔️     | ✔️               | ✔️                |
| Anthropic | ✔️     | ✔️               | ✔️                |
| Google    | ✔️     | ✔️               | ✔️                |
| Azure     | ✔️     | ✔️               | ✔️                |
| Custom    | ✔️     | User-defined     | User-defined      |

Adding in more providers is driven by customer demand. If you have a specific provider or model you would like to see supported, please reach out to us at [support@humanloop.com](mailto:support@humanloop.com).
