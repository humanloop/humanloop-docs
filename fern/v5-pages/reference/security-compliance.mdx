At Humanloop, we take security and compliance seriously. We understand that our customers trust us with their sensitive data and we are committed to maintaining the highest standards of security and data protection.

## Security Measures

### Data Encryption

- All data in transit is encrypted using TLS 1.2 or higher.
- Data at rest is encrypted using AES-256 encryption.

### Access Control

- We implement strict role-based access control (RBAC) for all our systems.
- Multi-factor authentication (MFA) is enforced for all employee accounts.

### Infrastructure Security

- Our infrastructure is hosted on AWS, leveraging their advanced security features.
- We use firewalls, intrusion detection systems, and regular security audits to protect our systems.

## Compliance

### SOC 2 Compliance

Humanloop is SOC 2 Type II compliant. This certification demonstrates our commitment to:

- Security
- Availability
- Processing Integrity
- Confidentiality
- Privacy

### GDPR Compliance

We are fully compliant with the General Data Protection Regulation (GDPR). This includes:

- Data minimization practices
- User rights management
- Data processing agreements

### HIPAA Compliance

For customers requiring HIPAA compliance, we offer:

- Dedicated instances with full [HIPAA compliant](https://aws.amazon.com/compliance/hipaa-compliance/) AWS setup.
- Business Associate Agreements (BAA) upon request.

## Data Protection

- We never train on customer data.
- Data residency options are available for customers with specific geographic requirements.

## Regular Audits and Penetration Testing

We conduct regular internal and external security audits and penetration tests to ensure the ongoing security of our systems.

## Reporting Security Concerns

If you believe you've found a security vulnerability in our service, please report it to security@humanloop.com. We take all reports seriously and will respond promptly.

For more detailed information about our security practices or to request our security documentation, please contact our sales team.
